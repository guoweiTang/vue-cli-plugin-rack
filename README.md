# cra-template-rack

This is a template for [Create React App](https://github.com/facebook/create-react-app).

## usage

```sh
$ npx create-react-app my-app --template rack
```
## Highlights
> Compared with create-react-app's official template cra-template
- sass
- axios
- router
- redux
- prettier
- lint
- mock
- test
- dockerFile
- stipulate directory
